# OpenCart Manual Testing Project

##  Project Overview

This repository contains a complete **Manual Testing** project for the **OpenCart E-Commerce Application**. The project follows the **Software Testing Life Cycle (STLC)** and demonstrates the process of planning, designing, executing, and reporting software tests.

The primary goal is to verify that the core functionalities of the OpenCart application work as expected while identifying and documenting defects through structured testing.

----------------------------------------------------------------------------------------------------------------------

# Objectives

* Validate the functionality of the OpenCart application.
* Design comprehensive test scenarios and test cases.
* Execute manual test cases.
* Identify and report software defects.
* Maintain proper testing documentation.
* Demonstrate industry-standard QA practices.
